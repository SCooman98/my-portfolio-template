This project is designed to be easily cloned and deployed by others. Follow these steps to create your own version:

### Step 1: Create Your Copy
1.  Navigate to the main page of this repository on GitHub.
2.  Click the green **"Use this template"** button (top right).
3.  Select **"Create a new repository"**.
4.  Give your new repository a name (e.g., `my-portfolio`).
5.  Click **"Create repository"**.

### Step 2: Deploy to Netlify
1.  Log in to your [Netlify](https://www.netlify.com/) account.
2.  Click **"Add new site"** > **"Import from Git"**.
3.  Choose **GitHub** as your Git provider.
4.  Select the repository you just created in Step 1.
5.  **Build Settings:**
    *   **Build command:** `npm run build`
    *   **Publish directory:** `dist`
6.  Click **"Deploy site"**.

### Step 3: Customize
Once deployed, you can clone your new repository to your local machine to edit the content:

```bash
git clone https://github.com/YOUR_USERNAME/YOUR_REPO_NAME.git
cd YOUR_REPO_NAME
npm install
npm run dev
```

Edit the text and images in the `src` folder to make it your own!
